## Layout posture

- Editorial luxury: generous whitespace, large serif headlines, refined minimalism
- Split hero layout: text left, image right bleeds edge-to-edge
- Warm beige canvas (`--surface`: #F5F0E8) as foundation; white (`--bg`) for contrast areas
- Single terracotta accent (`--accent`: #C4956A) — used for eyebrow, CTA borders, dividers
- No rounded cards; use borders + whitespace for structure
- Subtle warm gradient overlay on hero image for text legibility
- Hairline borders (1px), restrained shadows/none
